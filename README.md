# Alauda AI Landing Page

A modern, responsive landing page for Alauda AI.

## Features

- Responsive design
- Modern gradient hero section
- Feature showcase
- Contact form
- Smooth scrolling navigation

## Getting Started

Simply open `index.html` in your browser to view the page.

## Structure

- `index.html` - Main HTML file
- `styles.css` - Styling
- `README.md` - This file

## Customization

Replace the placeholder content in `index.html` with your actual:
- Company information
- Feature descriptions
- Contact details
- Images and logos
